# Quandl

Ruby wrapper for the Quandl API (www.quandle.com).

Supports:

- Datasets
- Metadata
- Multisets
- Searches
- Favorites

## Installation

Add this line to your application's Gemfile:

    gem 'quandl'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install quandl
